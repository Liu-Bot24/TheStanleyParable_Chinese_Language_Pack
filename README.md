# The Stanley Parable / 史丹利的寓言 简体中文汉化包

《The Stanley Parable》的简体中文汉化包，于游戏安装目录提取字幕和 UI 文本，按剧情上下文分组，使用 Gemini CLI 完成翻译和审校，并生成汉化包。

## 安装说明

从 GitHub Releases 下载发布用 ZIP 压缩包。

打开压缩包，将其中的 `thestanleyparable` 文件夹解压到《The Stanley Parable》游戏根目录，选择合并并覆盖同名文件。

## 文件说明

- `thestanleyparable`：汉化覆盖文件。
- `TheStanleyParable_CN_Overlay_20260615_zhcn_v3.zip`：最终发布用压缩包。
- 字体使用 Noto Sans SC，并随包附带 SIL Open Font License 许可证。

## 恢复原版

在 Steam 中对游戏执行“验证游戏文件完整性”，即可恢复被覆盖的原版文件。
